# 标准摄像机等设备Visio图标资源

## 资源描述

本仓库提供了一套用于Visio制图软件的标准摄像机及其他视频监控设备的图标资源。这些图标适用于设计视频监控系统图，包括但不限于以下设备类型：

- **半球摄像机**：适用于室内或室外监控，具有较小的外形尺寸。
- **球机摄像机**：具有360度旋转功能，适用于全方位监控。
- **专业摄像机**：适用于特殊环境或高要求的监控场景。
- **报警类产品**：如红外探头等，用于检测异常情况并触发报警。

## 使用说明

1. **下载资源**：点击仓库中的资源文件进行下载。
2. **导入Visio**：打开Visio软件，选择“文件” -> “形状” -> “更多形状” -> “打开形状库”，然后选择下载的图标文件进行导入。
3. **使用图标**：导入成功后，可以在Visio的形状库中找到这些图标，并将其拖拽到绘图区域进行使用。
